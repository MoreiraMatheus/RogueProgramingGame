# RogueProgramingGame

Um dos projetos para iniciantes em uma linguagem de programação é um rpg de texto para que o programador tenha conhecimentos básicos de variáveis e condicionais de código, pensando nisso decidi fazer o meu primeiro em C#.

Nesse projeto pude ter contato com os padrões de código utilizados em C# e conceitos inciiais de programação orientada a objetos
